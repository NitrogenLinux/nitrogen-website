Ready to install Nitrogen? Here's how to do so:

Step 1:

Download the latest Nitrogen Linux release from https://www.nitrogenlinux.com/downloads/.

Step 2:

Get a stick, you will need it.

Step 3:

Flash the ISO file on your stick using either

  • Rufus
    • or
  • Balena Etcher

    • Step 4:

    Get into your computer's BIOS, the key to do so differs from vendor to vendor(usually Esc).

    Step 5:

    Get into Nitrogen's Installation Environment.

    Step 6:

    Login as user root.

    Step 7:

    Type: "nitrogen-installer" and press enter.

    Continue configuring, from here the installation is simple.
