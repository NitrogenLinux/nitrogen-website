Installation Guide

So you've decided that you want to install Nitrogen Linux, how do you do it?

----------------------------------------------------------------------------

First of all partition your drive and mount root swap and efi(BIOS Systems can skip efi)

Then install lmt(Elements Package Manager) to /mnt

+---------------------------------------------------+
|git clone https://github.com/Tekq/lmt-installer.git|
|cd lmt-installer                                   |
|./install.                                         |
+---------------------------------------------------+
