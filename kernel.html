Linux Kernel

The Linux kernel is a core component in the GNU/Linux system, developed by finnish developer Linus Torvalds in 1991

-----------------------------------------

To install just do: "lmt --add linux-variation"

Note: change variation to whatever kernel you want(example: -nitrogen -zen -lts -vanilla)
