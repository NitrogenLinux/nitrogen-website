Coming 2022!

Nitrogen Kernel is our own kernel, developed by Nitrogen Project.

We want to make an UnBloated kernel which works on most hardware and is fast.

Happily you can have both Linux and Nitrogen Kernel on one machine.

Since Nitrogen Kernel will be backwards compatible all your Linux apps will work!

Nitrogen Kernel will not in any way replace Linux for a lot of time, so do not worry about that!
