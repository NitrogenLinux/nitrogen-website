Elements is Nitrogen Linux's official package manager.

It is created in Rust, and is well tested.

Rust allows Elements to have amazing Speed, that's why it is also very optimized for speed.

Elements' only limitation is the hardware it can run on, it is always tested on a x86-64-v2 or newer machine and is compiled with x86-64-v2 flag, that's why Elements might not run on older CPUs
